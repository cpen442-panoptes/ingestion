Welcome to Panoptes.

This is the web endpoint for DPI data submitted to Panoptes. Your can use a simple JSON API to submit packets obtained from libprotoident and nDPI.

Data ingestion API reference

POST https://calm-reef-40637.herokuapp.com/ingest/packet

  {
    "NodeIP": "192.168.0.2",
    "NodeID": "listener1",
    "Tool": "libprotoident",
    "SourceIP": "208.67.32.43",
    "SourcePort": 443,
    "DestinationIP": "192.168.0.6",
    "DestinationPort": 23143,
    "Type": "HTTPS"
  }

The server will reply with {"STATUS":"OK"} or {"STATUS":"ERR"}.

The following is an example call using curl:

curl -v -X POST https://calm-reef-40637.herokuapp.com/ingest/packet -H 'content-type: application/json' -d '{"NodeIP":"192.168.0.2","NodeID":"listener1","Tool":"libprotoident","SourceIP":"208.67.32.43","SourcePort":443,"DestinationIP":"192.168.0.6","DestinationPort":23143,"Type":"HTTPS"}'
